======
format
======

Description
===========
Returns a string containing the format (in struct module style) for each element in the view. 

Syntax
======
**memoryview**. *format*

Return Value
============
**str**

Time Complexity
===============
#TODO

Remarks
====
Defaults to 'B', a simple bytestring.

Example
=======
>>> m = memoryview('foo')
>>> m.format
'B'
