======
shape
======

Description
===========
Returns a tuple of integers the length of ndim giving the shape of the memory as a N-dimensional array.

Syntax
======
**memoryview**. *shape*

Return Value
============
**tuple**

Time Complexity
===============
#TODO

Example
=======
>>> m = memoryview('foo')
>>> m.shape
(3L,)
