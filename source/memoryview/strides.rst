======
strides
======

Description
===========
Returns a tuple of integers the length of ndim giving the size in bytes to access each element for each dimension of the array. 

Syntax
======
**memoryview**. *strides*

Return Value
============
**tuple**

Time Complexity
===============
#TODO

Example
=======
>>> m = memoryview('foo')
>>> m.strides
(1L,)
