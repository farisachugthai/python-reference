===
oct
===

Description
===========
Returns an integer converted into an octal string.

Syntax
======
**oct** *(integer)*

*integer*
	Required. An integer value to be converted into octal string.

Return Value
============
**str**

Example
=======
>>> oct(10)
'012'
>>> oct(256)
'0400'
>>> oct(4294967296L)
