=====
tuple
=====

Description
===========
Returns a tuple built from iterable.

Syntax
======
**tuple** *(iterable)*

*iterable*
	Optional. An iterable object.

>>> tuple((1, 2))
(1, 2)
>>> tuple({1, 2})
(1, 2)
>>> tuple({'a': 1, 'b': 2})
