======================
/= Division Assignment
======================

Description
===========
Divides the variable by a value and assigns the result to that variable.

Syntax
======
**A /= B**

*A*
    Any valid object.
*B*
    Any valid object.

Return Value
============
According to coercion rules.

Remarks
=======
Equivalent to A = A / B.	

Example
=======
>>> a = 10
>>> a /= 5
