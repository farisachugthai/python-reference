=========================
== is equal to
=========================

Description
===========
Returns a Boolean stating whether two expressions are equal.

Syntax
======
**A == B**

*A*
    Any valid object.
*B*
    Any valid object.

Return Value
============
**bool**

============

Example 1
=======
>>> 'ABC' == 'ABC'
True
>>> 1 == 1
True
>>> 1 == 1.0
True
>>> {1: None, 2: None} == 10
False

Example 2
====
>>> a = 1
>>> b = 1
>>> a == b
True
