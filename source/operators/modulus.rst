=========
% modulus
=========

Description
===========
Returns the decimal part (remainder) of the quotient.

Syntax
======
**A % B**

*A*
    Any expression evaluating to a numeric type.
*B*
    Any expression evaluating to a numeric type.

Return Value
============
According to coercion rules.
