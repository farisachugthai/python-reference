=========================
-= Subtraction Assignment
=========================

Description
===========
Subtracts a value from the variable and assigns the result to that variable.

Syntax
======
**A -= B**

*A*
    Any valid object.
*B*
    Any valid object.

Return Value
============
According to coercion rules.
