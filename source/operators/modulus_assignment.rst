=====================
%= Modulus Assignment
=====================

Description
===========
Computes the modulus of the variable and a value and assigns the result to that variable.

Syntax
======
**A %= B**

*A*
    Any valid object.
*B*
    Any valid object.

Return Value
============
According to coercion rules.

Remarks
=======
Equivalent to A = A % B.

Example
=======
>>> a = 5
>>> a %= 2
>>> a
