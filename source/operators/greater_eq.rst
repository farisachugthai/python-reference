======================================
>= greater than or equal to
======================================

Description
===========
Returns a Boolean stating whether one expression is greater than or equal the other.

Syntax
======
**A >= B**

*A*
    Any valid object.
*B*
    Any valid object.

Return Value
============
**bool**
