=======================
**= Power Assignment
=======================

Description
===========
Raises the variable to a specified power and assigns the result to the variable.

Syntax
======
**A **= B**

*A*
    Any valid object.
*B*
    Any valid object.

Return Value
============
According to coercion rules.
