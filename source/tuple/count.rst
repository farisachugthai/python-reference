=====
count
=====

Description
===========
Returns the number of times the specified item appears in the list.

Syntax
======
**list**. *count(item)*

*item*
    Required. Any valid type.

Return Value
============
**int**

Example
=======
>>> [1, 1, 2].count(1)

>>> [1, 1, 2].count(2)

>>> [1, 1, 2].count(5)
