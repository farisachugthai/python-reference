======
append
======

Description
===========
Adds an item to the end of the list.

Syntax
======
**list**. *append(object)*

*object*
    Required. Any valid type.

Return Value
============
**None**

Time Complexity
===============
O(1)

Example
=======
>>> l = [1, 2]
>>> l.append(3)
>>> l
[1, 2, 3]
