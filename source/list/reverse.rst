=======
reverse
=======

Description
===========
Reverses the elements of the list, in place.

Syntax
======
**list**. *reverse()*

Return Value
============
**None**

Time Complexity
===============
O(n)

Example
=======
>>> l = [1, 2, 3]
>>> l.reverse()
>>> l
[3, 2, 1]
