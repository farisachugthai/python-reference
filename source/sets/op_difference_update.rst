======================
-= (difference_update)
======================

Description
===========
Updates the set, removing elements found in the other set.

Syntax
======
set -= other | ...

*other*
    A set object or expression evaluating to a set.

Return Value
============
**set**

===============

Example
=======
>>> s = {1, 2}
>>> s -= {2, 3}
>>> s
set([1])
