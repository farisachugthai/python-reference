=================
!= (is not equal)
=================

Description
===========
Returns a Boolean stating whether the set has different elements as the other set.

Syntax
======
**set != other**

*other*
    A set object or expression evaluating to a set.

Return Value
============
**set**

Example
=======
>>> {0, 1} != {1, 0}

>>> {0, 1} != {1, 0, 2}
