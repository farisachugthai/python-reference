=======================================
Comprehensions and Generator Expression
=======================================
    
Comprehensions
==============
`[] list comprehension`_
    Returns a list based on existing iterables.
`{} set comprehension`_
    Returns a set based on existing iterables.
`{} dictionary comprehension`_
    Returns a dictionary based on existing iterables.

Generator Expression
====================
`() generator expression`_
    Returns an iterator over elements created by using list comprehension.
